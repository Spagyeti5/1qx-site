# 1qx Project - Local Setup Guide

1. Open the 1qx-project folder.
2. Double-click `start.bat` to launch the local server.
3. Your site will open at http://localhost:5173
4. If needed manually:
   - Open terminal inside folder.
   - Run `npm install`
   - Then run `npm run dev`
5. Play quizzes and rebus puzzles!

Use `check-node.bat` anytime to verify Node.js health.

Enjoy!
